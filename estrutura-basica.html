Estrutura Básica HTML

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável. Sempre será dessa forma e você sempre começará seu HTML começando por esse código. Geralmente os editores já tem atalhos para iniciar os documentos HTML com essa estrutura, logo, você não precisa se preocupar em decorá-la, mas é bom que faça. Veja abaixo como ela se inicia:



Tags HTML

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página. A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: <>. As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento. As tags que precisam de fechamento possuem a sintaxe , já as que não precisam de fechamento possuem como estrutura . Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.



TAG HEAD

O elemento HEAD providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.



TAG TITLE

O elemento HTML TITLE define o título do documento, mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.



TAG BODY

A tag BODY tem como finalidade definir o corpo do documento, ou seja, tudo que estiver dentro da tag body será mostrado de alguma forma no conteúdo da página.



TAGS H1, H2, H3, H4, H5, H6

Os elementos HTML H1-H6 representam seis níveis de título da seção, onde H1 é o nível mais alto e H6 é o nível mais baixo.



TAG P

A tag 'P' representa um parágrafo. Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada elemento 'P'.



TAG A

O elemento 'IMG' representa a inserção de imagem no documento. Navegadores nem sempre exibem a imagem referenciada pelo elemento. Este é o caso para navegadores não gráficos (incluindo aqueles usados por pessoas com deficiência de visão), ou se o usuário optar por não exibir imagens ou se o navegador não conseguir exibir a imagem porque é inválido ou um tipo não suportado. Nesses casos, o navegador pode substituir a imagem pelo texto definido no atributo alt deste elemento.



TAG BR

O elemento 'BR' produz uma quebra de linha em um texto. É útil para escrever poemas ou um endereço, onde a divisão de linha é significante. Não deve ser usado para para aumentar o espaço entre as linhas de texto; para isso devemos usar a propiedade CSS margin ou o elemento 'P'.
